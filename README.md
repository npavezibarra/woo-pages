# Woo Pages

A WordPress plugin to create new templates for WooCommerce pages.

## Description

This plugin allows you to customize and create new templates for WooCommerce pages, enhancing the flexibility of your store's design.

## Installation

1. Upload the plugin files to the `/wp-content/plugins/woo-pages` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Ensure WooCommerce is installed and active.
